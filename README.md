# Vehicle Kick (open.mp/sa-mp)

## Description
Simple vehicle interaction feature for **open.mp/sa-mp**.  
Allows players to kick the nearest vehicle with animation, sound, and physical reaction.

---

## Command
### `/kickveh`
Kicks the closest vehicle within **3.0 units**.

---

## Features
- Detects nearest vehicle automatically
- Adjusts player facing direction
- Plays kick animation
- Applies vehicle angular velocity
- Includes sound effect

---

## Notes
- Visual / roleplay feature
- No damage system included
- Fully customizable (range, animation, force)
